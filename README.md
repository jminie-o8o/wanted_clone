## 🔎 핵심 기능 및 담당 기능
원티드 서비스의 핵심기능은 채용정보 제공 및 구직자와 구인자의 매칭입니다.
이러한 점에서 저희는 많은 원티드 서비스 중에서도 회원과 기업, 채용정보, 이력서 기능에 초점을 맞춰 개발했습니다.
저는 그 기능들 중에서도 `회원가입, 로그인, 회원정보, 채용정보, 이력서 생성`과 관련된 API를 담당하였습니다.
